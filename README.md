# Ancestral Surf AI

Sistema de autoconocimiento basado en:

- conocimientos ancestrales
- vida costera
- surf
- pesca artesanal
- aprendizaje automático

La app registra sensaciones personales y aprende patrones.

## Variables registradas

energía
ánimo
estrés
sueño
surf
pesca
cannabis
alimentación

## Características

fase lunar
clima
recomendaciones
aprendizaje IA
visualización de datos

## ejecutar

pip install -r requirements.txt

streamlit run app.py
